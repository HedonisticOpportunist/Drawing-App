General Button Overview

The buttons referred to here are displayed at the top of the application.

  • The save button saves your drawing as a PNG/png file.
  • The clear button clears the canvas.
  • The 'reload application' button reloads the application in case the canvas freezes or certain tools do not work.

Colour Palette Overview

Upon starting the application, the colour will be set to random. Otherwise use the colour palette to change colours.

Use the slider at the top of the application to change the stroke weight.

Tools Overview

    The drawing app supports the following tools:
  • A freehand tool that lets you draw on the canvas.
  • A draw line tool that allows you to draw lines on the canvas.
  • A spray can tool that lets you spray on the canvas.
  • A mirror draw tool that lets you draw items that are displayed as mirror reflections on the canvas.
  • A transparent layer tool that lets you draw over an existing sketch with a transparent colour of your choice (via the colour palette).
  • An image tool that lets you place images on the canvas. Select the image you want displayed from the bottom of the application.
  • A shape tool that lets you draw shapes on the canvas. Select the 2D shape you want displayed from the bottom of the application. You can use the 'fill shape with random colours' button to display a colour that is different from the colour palette when clicking on it.
  • A stamp tool that lets you draw stamps on the canvas. Select the stamp you want displayed from the bottom of the application. Use the slider to change the size of the stamp.
  • An eraser tool that allows you to erase items on the canvas. Use the slider to adjust the stroke weight of the eraser.
  • A random emoji string tool which displays random cat emojis on the screen. Use the slider to adjust the size of the text and the colour palette to change the stroke colour surrounding the text.
  • A 3D tool that displays a cone and / or sphere on the canvas. Please note that there is a limit to how much this tool can be used. Reload the application if you want to create a new 3D image.
  • An animation tool which lets you draw various animations on the canvas. Choose the animation you want displayed from the bottom of the application. Use the slider to either speed up or slow down the relevant animation.
  • A cropping tool lets you cut the area of your choice by drawing a white rectangle over it.
