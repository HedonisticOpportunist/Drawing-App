EXTENSIONS

3D TOOL

The 3D tool creates buffer graphics which are saved as an image to the canvas; the buffer includes the WEBGL canvas which is created separately upon instantiation. The canvas for the 3D drawing is 400 by 400, and its background colour can be changed via the colour palette. In order to avoid memory leaks, the 3D tool imposes limits on how many sketches can be drawn before reloading the application.

ANIMATION TOOL

The animation tool can either draw sprites or load images as a sequence of animations on the canvas. There are three available animations to choose from. Upon selection of an animation, a compound boolean condition is triggered that then selects an event handler from the helper function in order to display the animation on the canvas.

CROPPING TOOL

The cropping tool selects an area via the mouse coordinates and then draws a beige background on the screen once the cut button has been clicked. The mouse click event is handled in the helper function.

ERASER TOOL

The eraser tool draws a line over a selected area via mouse coordinates. The stroke weightis modified via the slider. The colour of the eraser is the same as the background colour, unless the screen has been cleared or the console selected.

GEOMETRIC SHAPES TOOL

Geometric shapes can be selected from the images displayed in the div options field. Upon selection, the position of the shape is determined via mouse coordinates. The stroke weight of the shape is set by the slider. The colour of the shape can either be selected via the colour palette or by clicking on a button that displays a random hex colour.

ADD TRANSPARENT LAYER TOOL

The tool uses the blend mode and colour palette to draw a transparent rectangle over the canvas.

IMAGE TOOL

The image tool displays a selected image via a mouse click event in the helpers function. The event is triggered by clicking on one of the images displayed in the options div field.

RANDOM STRING TOOL

The random string tool picks a random cat emoji from an array and displays that emoji on the canvas. The stroke weight is modified via the slider. The background surrounding the text can be changed via the colour palette.

STAMP TOOL

The stamp tool displays a selected image via a mouse click event in the helpers function. The event is triggered by clicking on one of the image options displayed in the options div field. The stamp can be resized via the slider.

EVALUATION

UNIT TESTING / PERFORMANCE TESTING

Apart from a myriad of bugs that I found while testing (and fixed later on), performance issues when using the normal version of Chrome (not Google Canary) cropped up. For example, extensions containing images would stop working properly. In order to address these issues, I minified the external Javascript libraries which were creating a lot of noise.

USER TESTING

Generally, the overwhelming consensus of the users was that the application looked good and that the features were fun to explore. However, I received constructive criticism as well, some of which included:

  • The application is not necessarily self-explanatory as there are no default settings for the individual extensions.
  • The application is not responsive as certain extensions do not load on mobile or very large screens.
  • Some of the images -- like the cat stamp ones -- did not load properly.
  • The 3D feature was hard to use and caused memory leaks.

Most of the issues identified by users were ratified during the next development process and also inspired me to refactor the code as I went along. Still, I do not think that I have been able to address the responsiveness issue of the application as well as I would like -- after all, this would include an entire overhaul of the application
