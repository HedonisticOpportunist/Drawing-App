Another piece of css, I modified slightly is the code for the horizontal tab on the icon credits and help pages. The css code came from: https://css-tricks.com/examples/hrs/ The slider was implemented using: https://www.w3schools.com/howto/howto_js_rangeslider.asp which I also customised to further fit in with the rest of the application’s appearance. == JS CODE == In order to generate hex colours, I implemented a getRandomColour method in the drawingHelpers file. The code was found here: https://stackoverflow.com/questions/1484506/random-color-generator In order to minify the external Javascript libraries, I used https://jscompress.com/
